# Geração de Conteúdo

<div class="chart-container" style="position: relative; height:250px; width:250px; margin: 20px auto;">
    <canvas class="accessibility-pie-chart"></canvas>
</div>

Para produzir conteúdo digital acessível, a primeira etapa é compreender que tudo o que comunicamos impacta as pessoas; por isso, é fundamental começar pela forma como abordamos nosso público e criar a cultura e o hábito de acessibilidade digital em nossas equipes". Nesse sentido, nesta seção se apresentam boas práticas em formato de checklist para garantir acessibilidade na geração de conteúdo.

## Termos e Linguagem
Sempre utilize o termo "pessoas com deficiência" (PCD) e depois complemente: visual, física, intelectual, múltipla, pessoa surda, pessoa cega, usuária de cadeira de rodas (cadeirante), tetraplégica, paraplégica, pessoa com nanismo, com baixa visão, pessoa autista, disléxica, neurodiversa ou neurodivergente.

- [ ] Evite jargões e termos técnicos sem explicação. <a id="TEC1" href="#RP1">[1] [AAA]</a> <a id="NBR1" href="#NBRREF">[NBR 5.12.12]</a>

- [ ] Forneça definições para palavras incomuns ou técnicas. <a id="TEC2" href="#RP2">[2] [AAA]</a> <a id="NBR2" href="#NBRREF">[NBR 5.12.10]</a>

## Textos
- [ ] Use linguagem clara e simples. <a id="TEC1" href="#RP1">[1] [AAA]</a> <a id="NBR3" href="#NBRREF">[NBR 5.12.12]</a>

- [ ] Usar pontuação adequada. <a id="NBR4" href="#NBRREF">[NBR 5.12.12]</a>

- [ ] Evitar frases extensas (ideal é ter de 15 a 20 palavras). <a id="TEC1" href="#RP1">[1] [AAA]</a> <a id="NBR5" href="#NBRREF">[NBR 5.12.12]</a>

- [ ] Usar preferencialmente a ordem direta nas orações. <a id="NBR6" href="#NBRREF">[NBR 5.12.12]</a>

- [ ] Fornecer explicações para abreviações e siglas. <a id="TEC3" href="#RP3">[3] [AAA]</a> <a id="NBR7" href="#NBRREF">[NBR 5.12.11]</a>

- [ ] Estruturar conteúdo com títulos e subtítulos claros. <a id="TEC4" href="#RP4">[4] [AA]</a>

## Descrição de Imagens
- [ ] Fornecer texto alternativo para todas as imagens informativas(Tipo de imagem, localização, descrição e referência). <a id="TEC5" href="#RP5">[5] [A]</a> <a id="NBR10" href="#NBRREF">[NBR 5.2.1]</a> <a id="NBR11" href="#NBRREF">[NBR 5.2.2]</a>

- [ ] Marcar imagens decorativas com alt="". <a id="TEC5" href="#RP5">[5] [A]</a> <a id="NBR12" href="#NBRREF">[NBR 5.2.3]</a>

## Descrição de Imagens Complexas
- [ ] Posicionar gráfico, quadrinhos, quadros e outras imagens complexas em um local separado. <a id="NBR13" href="#NBRREF">[NBR 5.2.4]</a>

- [ ] Descrição sucinta no texto alternativo. <a id="TEC5" href="#RP5">[5] [A]</a> <a id="NBR14" href="#NBRREF">[NBR 5.2.4]</a>

- [ ] Fornecer uma descrição longa detalhada em texto ou em uma página separada. <a id="TEC6" href="#RP6">[6] [AAA]</a> <a id="NBR15" href="#NBRREF">[NBR 5.2.4]</a>

## Ícones
- [ ] Adicionar iconografia. <a id="NBR16" href="#NBRREF">[NBR 5.13.9]</a>

- [ ] Adicionar ícone e texto. <a id="NBR17" href="#NBRREF">[NBR 5.13.9]</a>

- [ ] Adicionar texto alternativo para ícones clicáveis. <a id="TEC5" href="#RP5">[5] [A]</a> <a id="NBR18" href="#NBRREF">[NBR 5.2.2]</a>

- [ ] Descrever no texto alternativo, quando necessário usá-lo, a ação da pessoa usuária e para onde será direcionada. <a id="NBR19" href="#NBRREF">[NBR 5.2.2]</a>

## Mídias de Vídeo
- [ ] Adicionar descritivo curto para vídeos. <a id="NBR20" href="#NBRREF">[NBR 5.14.2]</a>

- [ ] Optar por legendas: Open Caption e Closed Caption para vídeos. <a id="TEC7" href="#RP7">[7] [A]</a> <a id="NBR21" href="#NBRREF">[NBR 5.14.2]</a>

- [ ] Fornecer audiodescrição para vídeos. <a id="TEC8" href="#RP8">[8] [AA]</a>

- [ ] Fornecer alternativa em texto para vídeos (transcrição). <a id="TEC9" href="#RP9">[9] [AAA]</a>

- [ ] Fornecer interpretação em Libras para conteúdo pré-gravado. <a id="TEC10" href="#RP10">[10] [AAA]</a>

## Podcasts e Áudio
- [ ] Transcrever o conteúdo do episódio. <a id="TEC9" href="#RP9">[9] [AAA]</a> <a id="NBR25" href="#NBRREF">[NBR 5.14.1]</a> <a id="NBR26" href="#NBRREF">[NBR 5.14.3]</a>

- [ ] Incluir além das partes faladas, como sons de fundo, efeitos sonoros, quem está falando etc. <a id="TEC9" href="#RP9">[9] [AAA]</a>

- [ ] Inserir um avatar na página para a interpretação do texto transcrito para Libras.

- [ ] Considerar também formato de vídeo.

- [ ] Fornecer controle de reprodução para áudio que toca automaticamente. <a id="TEC11" href="#RP11">[11] [A]</a>

## Hashtags e Emojis
- [ ] Em hashtags, utilize a primeira letra de cada palavra em maiúsculas para que leitores possam identificar palavras corretamente.

- [ ] Não abusar de emojis, pois nem sempre sua descrição é suficiente para um bom entendimento. <a id="NBR33" href="#NBRREF">[NBR 5.12.12]</a>

- [ ] Fornecer alternativas textuais para emojis quando usados para transmitir informação. <a id="NBR34" href="#NBRREF">[NBR 5.2.1]</a>

- [ ] Quando cabível, utilizar a #paratodosverem

## Hiperlinks
- [ ] Descrição de links e botões devem ser compreensíveis de maneira isolada. <a id="TEC12" href="#RP12">[12] [A]</a> <a id="NBR36" href="#NBRREF">[NBR 5.7.3]</a> <a id="NBR37" href="#NBRREF">[NBR 5.7.4]</a>

- [ ] Indicar quando links abrem em nova janela. <a id="NBR38" href="#NBRREF">[NBR 5.7.6]</a>

## Conteúdo Dinâmico
- [ ] Fornecer alternativas para conteúdo que se move, pisca ou rola automaticamente. <a id="TEC13" href="#RP13">[13] [A]</a>

- [ ] Permitir que usuários controlem o tempo para ler ou usar conteúdo. <a id="TEC14" href="#RP14">[14] [A]</a> <a id="NBR40" href="#NBRREF">[NBR 5.16.2]</a>
